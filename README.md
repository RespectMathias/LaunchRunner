# Launch Runner

JetBrains-style run/debug buttons for VS Code.

## Install

Install from the [VS Code Marketplace](https://marketplace.visualstudio.com/vscode).

## Build

```bash
npm install
npx vsce package
```

## Usage

Click **...** to select a launch configuration or task. Use **▶️ Run** or **🐞 Debug** buttons to execute.

## Requirements

VS Code 1.80.0+ with `.vscode/launch.json` or `.vscode/tasks.json` in your workspace.
